📅 対象期間

各週の月曜日の日付を入れてください。曜日の見出しに日付が表示されます。

① 医師スケジュール

セルをタップして設定します。○=稼働 / ◎=処置あり / 契=契約 / ✓=物品庫チェック は複数選択できます（×=休のみ単独）。契約の日は結果表で【契】が付き、契約専任ドライバー（管理ページ2）だけが割り当てられます。物品庫チェックの日は、医師名の横に✓が付きます。

← 表は横にスクロールできます →

② 看護師

セルをタップすると選択メニューが開きます（○=稼働 / ×=休 / △=内勤）。PAで始まる名前はアシスタント扱い。

← 表は横にスクロールできます →

③ ドライバー

セルをタップすると選択メニューが開きます（◯=稼働 / △=内勤 / 単独 / ×=休）。1週目・2週目それぞれ設定できます。◯の人が各車の運転担当に割り当てられ、△＝内勤・単独は結果表の下の「I 内勤 / J 単独」に表示されます。
