📅 対象期間

各週の月曜日の日付を入れてください。曜日の見出しに日付が表示されます。

① 医師スケジュール

セルをタップして設定します。○=稼働 / ◎=処置あり / 契=契約 / ✓=物品庫チェック / 前=午前のみ / 後=午後のみ は複数選択できます（×=休のみ単独）。契約の日は結果表で【契】が付き、契約専任ドライバーだけが割り当てられます。物品庫チェックの日は、医師名の横に✓が付きます。前／後は表示ラベルで、割当は終日1台のまま（結果表に「午前」「午後」と表示）。

② 看護師

セルをタップすると選択メニューが開きます（○=稼働 / 前=午前のみ / 後=午後のみ / ×=休 / △=内勤）。前／後は表示ラベルで、割当は終日ベースのまま（結果表に「午前」「午後」と表示）。

← 表は横にスクロールできます →

③ ドライバー

セルをタップすると選択メニューが開きます（◯=終日稼働 / 前=午前のみ / 後=午後のみ / △=内勤 / 単独 / ×=休）。1週目・2週目それぞれ設定できます。稼働の人が各車の運転担当に割り当てられ、午前と午後は別々に割り当てます（原則は終日同じ人。不在時のみ交代）。△＝内勤・単独は結果表の下の「I 内勤 / J 単独」に表示されます。

← 表は横にスクロールできます →
