📅 対象期間

説明

カレンダーでいずれかの日を選ぶと、その週の月曜日から2週間を自動で表示・編集できます（何曜日を選んでもその週の月曜にそろえます）。日付を変えると別の2週間に切り替わり、それぞれ独立して作成できます（期間の上限なし・無期限）。触れた期間は「☁ この期間を保存」でまとめて共有保存され、「保存済みの期間…」から呼び出せます。

① 医師スケジュール

説明

セルをタップして設定します。○=稼働 / ◎=処置あり / 契=契約 / ✓=物品庫チェック / PD / 前=午前のみ / 後=午後のみ は複数選択できます（×=休のみ単独）。PDの日はPD対応ドライバーだけが割り当てられます（③で設定）。契約の日は結果表で【契】が付き、契約専任ドライバーだけが割り当てられます。物品庫チェックの日は、医師名の横に✓が付きます。前／後は表示ラベルで、割当は終日1台のまま（結果表に「午前」「午後」と表示）。

← 表は横にスクロールできます →

② 看護師

説明

セルをタップすると選択メニューが開きます（○=稼働 / 前=午前のみ / 後=午後のみ / ×=休 / △=内勤）。前／後は表示ラベルで、割当は終日ベースのまま（結果表に「午前」「午後」と表示）。

← 表は横にスクロールできます →

③ ドライバー

説明

セルをタップすると選択メニューが開きます（◯=終日稼働 / 前=午前のみ / 後=午後のみ / △=内勤 / 単独 / ×=休）。稼働の人が各車の運転担当に割り当てられ、午前と午後は別々に割り当てます（原則は終日同じ人。不在時のみ交代）。△＝内勤・単独は結果表の下の「I 内勤 / J 単独」に表示されます。2週間ぶん、日ごとに設定できます。

← 表は横にスクロールできます →

待機ドライバーの手動組み合わせ（医師 × ドライバー）
説明

特定の医師にこのドライバーを充てたいときに使います（日ごとに設定）。割当の優先順位は契約 ＞ 物品チェック ＞ 待機ドライバー。契約車・物品チェック車が先に確保され、待機の指名はそのルール内で反映されます（契約や物品チェックで必要な人は待機指名より優先）。
